# Snow CLI 使用文档——首次配置

欢迎使用 Snow CLI！这是一个功能强大的 AI 驱动命令行工具。

## 首次配置

### 1、在任意目录下进入命令行

1. 键入 `snow` 启动 Snow CLI 或点击IDE插件中的启动图标
2. Snow CLI的首选语言为 `English` 可先前往 `Language Settings` 中修改自己语言偏好

   ![alt text](images/image4.png)

### 2、开始配置AI API

1. 设置完语言偏好后进入 `API 和模型设置`
   ![alt text](images/image5.png)

2. Snow CLI 支持4种请求方案

   ![alt text](images/image6.png)

   - OpenAI Chat Completion
   - OpenAI Responses
   - Gemini
   - Anthropic

   **注意事项**

   1. OpenAI Chat Completion Base Url
      ```
      https://api.openai.com/v1
      ```
   2. OpenAI Responses Base Url
      ```
      https://api.openai.com/v1
      ```
   3. Gemini Base Url 
      ```
      https://generativelanguage.googleapis.com/v1beta
      ```
   4. Anthropic Base Url
      ```
      https://api.anthropic.com/v1
      ```
   5. 该列表为滚动列表，所有项都需要填写

   6. 最大上下文令牌为该模型的上下文容量token数

   7. 最大令牌数为该模型支持的响应token数
