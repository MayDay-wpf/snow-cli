# Snow CLI User Documentation - Proxy and Browser Settings

Welcome to Snow CLI! This is a powerful AI-driven command-line tool.

## Proxy and Browser Settings

* When proxy is enabled, CLI traffic will be transmitted through the custom port

* Browser Settings: The CLI's web search functionality will use a browser. By default, it selects the default browser for macOS and Windows. If the default browser's installation location has changed, please manually specify the browser path
