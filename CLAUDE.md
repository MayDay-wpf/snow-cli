# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Development Commands

- **Build**: `npm run build` - Compiles TypeScript to dist/
- **Watch mode**: `npm run dev` - Runs TypeScript compiler in watch mode
- **Test**: `npm run test` - Runs full test suite (prettier, xo linter, and ava tests)
- **Individual tools**:
  - Linting: `npx xo` (ESLint with XO config)
  - Formatting: `npx prettier --check .`
  - Tests only: `npx ava`

## Architecture

This is a CLI application built with:
- **React + Ink**: Terminal UI framework for React components
- **TypeScript**: Main development language, compiled to dist/
- **Meow**: CLI argument parsing and help generation
- **Entry point**: source/cli.tsx contains the main CLI setup
- **Main component**: source/app.tsx contains the React component logic

### Project Structure

- `source/`: Main application code (TypeScript source)
  - `cli.tsx`: CLI entry point with meow setup
  - `app.tsx`: Main React component
- `src/`: Additional modules (currently empty folders for components, types, utils)
- `dist/`: Compiled JavaScript output (generated by build)

### Development Notes

- Uses ES modules (`"type": "module"` in package.json)
- TypeScript config extends @sindresorhus/tsconfig
- Linting configured with XO + React rules, Prettier integration
- Tests use AVA with ts-node ESM loader
- Node 16+ required
